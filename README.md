## Giới thiệu

Khóa học này hướng dẫn sinh viên làm chủ cả lý thuyết đằng sau các thách thức kỹ thuật phần mềm truyền thống và cách áp dụng các công cụ AI hiện đại để giải quyết chúng.

### Chủ đề chính:
- Coding Agents và LLMs
- Model Context Protocol (MCP)
- AI IDE và công cụ phát triển
- Testing và Security với AI
- DevOps và Observability
